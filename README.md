# ignite-reactjs-desafio-07-upload-de-imagens

Desafio do curso Ignite - Rocketseat trata-se de um projeto de upload de imagens a partir de um template usando basicamente as seguintes tecnologias:
* API do Next.js;
* React Query;
* React Hook Form;
* ImgBB;
* FaunaDB;
* Figma.
